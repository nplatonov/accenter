
## R-package `ursa`

Non-Interactive Spatial Tools for Raster Processing and Visualization
(Platonov, [2018](#ref-r-ursa)).

This is suggested tool.

### Install

``` r
devtools::install_github("nplatonov/ursa")
```

### Use

``` r
require(ursa)
display(ursa_dummy(3))
if (interactive())
   ?ursa::about
```
